Frequently Asked Question

Radiation Tool Set is a WWW based set of engineering tools providing design engineers and scientists necessary analytical tools to evaluate electronic components for radiation effects on space, avionics, networks, and medical devices. With the Radiation Tool Set, you can analyze SEU data, Fit proton test data to a Weibull, fit heavy ion data to the 7-parameter HICUP model, look up radiation data on a large collection of parts database

An important feature of Radiation Tool Set is the possibility to add more tools that is necessary for radiation analysis and assessment. Another feature is the Radiation Tool Set working environment. Each user has a permanent personal account. This approach ensures that the inputs to the models and the results of the computations are not lost when leaving Radiation Tool Set. Comparison of different runs is easy; thanks to the possibility to define several projects, each characterized by its own set of inputs.
