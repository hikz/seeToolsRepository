Frequently Asked Question

SEE Tool Set is a web based set of engineering tools providing design engineers and scientists necessary tools to evaluate electronic components for radiation effects on space, avionic, networks, and medical devices. With SEE Tool Set, you can analyze SEU data and generate HICUP 7-parameters for SEU rate prediction

 An important feature of SEE Tool Set is the possibility to add more tools that is necessary for SEU assessment. Another feature is the SEE Tool Set working environment. Each user has a permanent personal account. This approach ensures that the inputs to the models and the results of the computations are not lost when leaving SEE Tool Set. Comparison of different runs is easy thanks to the possibility to define several projects, each characterized by its own set of inputs. Please consult the Terms and Conditions, here You can use the bottom tool bar included on all the SEE Tool Set pages to either report a bug, access the SEE Tool Set forum or contact the SEE Tool Set team In order to use the SEE tools on this server, you do not need any additional software, just a client application such as FireFox, Internet Explorer or Safari. SEE Tool Set is known to work fine on a wide variety of browsers.
